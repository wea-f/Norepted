About Norepted

A YouTube unblocker built for school & work environments.

What is it?

Norepted embeds YouTube videos using the same method a teacher uses to embed videos in Google Slides — the official embed URL. It extracts your video ID and loads it in an iframe with no YouTube interface, no ads, and no browsing history.

GitHub Repository →

Features

  • ✓ Limited* ads in embeds
  • ✓ YouTube search & Shorts scroller
  • ✓ Multi-tab proxy browser
  • ✓ Tab cloaking (looks like Google Drive)
  • ✓ History wipe shortcut
  • ✓ Saves your open videos across sessions
